# 📱 Aplicativo de Área do Aluno Universitário

## 📌 Descrição do Projeto
Este projeto consiste no desenvolvimento de um **wireframe mobile de baixa fidelidade**, criado no Figma, para um aplicativo de **área do aluno universitário**.

O objetivo é propor uma solução digital que facilite o acesso às principais funcionalidades acadêmicas, melhorando a experiência do estudante em sua rotina universitária.

---

## 🎯 Problema
Estudantes universitários frequentemente enfrentam dificuldades para acessar informações acadêmicas de forma rápida e organizada.

---

## ✅ Objetivo
Desenvolver um fluxo de interface que permita ao aluno realizar sua principal tarefa de forma simples, clara e eficiente, garantindo:

- Clareza funcional  
- Boa arquitetura da informação  
- Flexibilidade de uso  
- Experiência intuitiva  

---

## 👤 Tarefa Principal do Usuário
A principal tarefa definida para o usuário é:

> **Acessar informações acadêmicas de forma rápida e organizada.

---

## 🔄 Fluxo do Usuário
O protótipo apresenta **dois caminhos distintos** para a realização da mesma tarefa:

### Caminho 1:
- Tela inicial (dashboard)
- Acesso direto às disciplinas
- Visualização de informações (notas/horários)

### Caminho 2:
- Menu de navegação
- Seleção da opção desejada
- Acesso às mesmas informações

---

## 🧠 Arquitetura da Informação
A estrutura do aplicativo foi organizada para facilitar a navegação, contendo:

- Tela inicial (resumo geral)
- Disciplinas
- Avisos
- Atividades

---

## 🎨 Protótipo
O wireframe foi desenvolvido no Figma com foco em baixa fidelidade, priorizando:

- Estrutura da interface  
- Organização das informações  
- Navegação clara

## 🧪 Testes de Usabilidade
Foram realizados testes com usuários para validar:

- Facilidade de uso  
- Clareza das telas  
- Eficiência na realização da tarefa  

Os testes ajudaram a identificar melhorias no fluxo e na organização das informações.
